# Library Management System

## Overview

This project is a simple console-based Library Management System developed in Java as part of the B202 Advanced Programming module. The application demonstrates the core principles of Object-Oriented Programming (OOP), including encapsulation, inheritance, polymorphism, and method overriding.

Users can view all books, add new books, borrow available books, and return borrowed books through an interactive menu. The system stores books using Java's ArrayList, making it easy to manage the library collection dynamically.

### Features

* Add standard and academic books
* Display all books
* Borrow and return books
* Console-based user interface
* Object-Oriented Programming design

### Technologies

* Java
* Object-Oriented Programming (OOP)
* Java Collections (ArrayList)
* IntelliJ IDEA / Eclipse

This project was developed for educational purposes to demonstrate Java programming fundamentals and software design principles.
